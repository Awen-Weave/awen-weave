# Contributing to Awen Weave

Thank you for considering a contribution.

## CLA

External code contributions will require a Contributor License Agreement (CLA) once the template is finalised. The template is being prepared as part of the commercial license documentation work; expected mid-2026. Until then, contributions are paused for external contributors.

In the interim:
- Bug reports and feature requests via GitHub Issues are welcome.
- Documentation suggestions via Issues are welcome.
- Pull requests will be acknowledged but held until the CLA process is in place.

## Internal contributions

For Awen Weave Limited maintainers: standard PR flow per working-patterns conventions. STATUS.md update + commit message + branch off main + PR template.
